## AI Agent Frameworks

- [**AutoGen**](https://github.com/microsoft/autogen)  -  Microsoft's multi-agent conversation framework for building complex AI systems with role-based agents
- [**CrewAI**](https://github.com/joaomdmoura/crewai)  -  Framework for orchestrating autonomous AI agents that work together as a crew
- [**Langchain**](https://github.com/langchain-ai/langchain)  -  Foundational library for building applications with LLMs, offering chains, agents, and memory abstractions
- [**Open Interpreter**](https://github.com/OpenInterpreter/open-interpreter)  -  Let LLMs run code locally and interact with your computer

## Code & Development

- [**Auto-GPT**](https://github.com/Significant-Gravitas/Auto-GPT)  -  Early autonomous AI agent that can break down goals and execute them iteratively
- [**Aider**](https://github.com/Aider-AI/aider)  -  AI pair programmer that can edit code in your local repository
- [**Prompt Engineering Guide**](https://github.com/dair-ai/Prompt-Engineering-Guide)  -  Comprehensive guide with papers, techniques, and best practices

## Specialized Tools

- [**OpenClaw**](https://github.com/openclawcompute)  -  AI agent framework for operating graphical user interfaces directly
- [**Ollama**](https://github.com/ollama/ollama)  -  Simple way to run large language models locally
- [**LiteLLM**](https://github.com/BerriAI/litellm)  -  Unified interface for calling all major LLM APIs with cost tracking

## Research & Resources

- [**Transformers**](https://github.com/huggingface/transformers)  -  Hugging Face's comprehensive library for state-of-the-art NLP models
- [**Papers with Code**](https://github.com/paperswithcode/paperswithcode-data)  -  Curated dataset linking papers with their implementations
