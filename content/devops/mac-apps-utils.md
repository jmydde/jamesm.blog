## Productivity & Writing
- [Microsoft 365](https://www.microsoft.com/en-ie/microsoft-365/mac/microsoft-365-for-mac) 💰  -  Suite of office applications
  - [Excel](https://www.microsoft.com/en-ie/microsoft-365/excel)  -  Spreadsheet application
  - [OneNote](https://www.microsoft.com/en-ie/microsoft-365/onenote/digital-note-taking-app)  -  Digital note-taking
  - [Outlook](https://www.microsoft.com/en-ie/microsoft-365/outlook/email-and-calendar-software-microsoft-outlook)  -  Email & calendar management
  - [Powerpoint](https://www.microsoft.com/en-ie/microsoft-365/powerpoint)  -  Presentation software
  - [Word](https://www.microsoft.com/en-ie/microsoft-365/word)  -  Document writing
- [Notion](https://www.notion.so/) 💰  -  All-in-one workspace for notes, databases, and project management
- [Obsidian](https://obsidian.md/) 💰  -  Private markdown-based writing and knowledge management app
- [MindNode](https://www.mindnode.com/) 💰  -  Mind mapping and brainstorming tool

## Development & Version Control
- [PyCharm](https://www.jetbrains.com/pycharm/) 💰  -  Comprehensive Python IDE with debugging and testing tools
- [GitKraken](https://www.gitkraken.com/) 💰  -  Powerful visual Git client with integrated workflows
- [iTerm2](https://iterm2.com/) 🆓  -  Advanced terminal emulator with split panes and extensive customization
- [FileZilla](https://filezilla-project.org/) 🆓  -  FTP, FTPS & SFTP client for file transfer
- [Kaleidoscope](https://kaleidoscope.app/) 💰  -  Visual diff tool for comparing text, images, and folders
- [VisualDiffer](https://visualdiffer.com/) 💰  -  Advanced folder & file comparison utility

## File & System Management
- [Path Finder](https://www.cocoatech.io/) 💰  -  Advanced file manager with extended functionality
- [The Unarchiver](https://theunarchiver.com/) 🆓  -  Open any archive format (ZIP, RAR, 7z, etc.)
- [CleanMyMac](https://cleanmymac.com/) 💰  -  System cleanup and optimization utility
- [Disk Space Analyzer Pro](https://apps.apple.com/us/app/disk-space-analyzer-pro/id488920185) 💰  -  Visualize and reclaim disk space
- [DiskCatalogMaker](https://diskcatalogmaker.com/) 💰  -  Create and manage disc catalogs
- [DirEqual](https://apps.apple.com/us/app/direqual/id1435575700) 💰  -  Compare and sync folder contents
- [CloudMounter](https://cloudmounter.net/) 💰  -  Mount cloud storage (Dropbox, Google Drive, OneDrive, S3) as local drives
- [Google Drive](https://apps.apple.com/au/app/google-drive/id507874739) 🆓  -  Cloud storage and file sync
- [OneDrive](https://apps.apple.com/us/app/onedrive/id823766827) 🆓  -  Microsoft cloud storage integration

## Text & Data Tools
- [TextSoap](https://textsoap.com/mac/) 💰  -  Batch text transformations and cleanup
- [Text Workflow](https://www.gtrigonakis.com/textworkflow) 💰  -  Automation engine for text manipulation
- [Easy Data Transform](https://www.easydatatransform.com/) 💰  -  Data merging, splitting, cleaning without coding
- [Pure Paste](https://sindresorhus.com/pure-paste) 💰  -  Clipboard manager that pastes as plain text by default
